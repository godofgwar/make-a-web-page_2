Markup is a set of markup symbols or codes inserted in a file intended for display on a browser.

Inline vs. Block Elements

Inline elements don't need a closing tag and are displayed without line breaks. Block elements need to be closed and create an imaginary box around the line

For example, br is inline and creates a new line

p (for paragraph) is block - makes an invisible box around text.

span is inline

div is a block element

A Complete HTML Document

A complete HTML document should contain the following:

  1. The doctype, e.g., HTML
  2. Tags
  3. "head" which includes metadata, css and more
  4. "title" which is the title of the page at the top of the browser
  5. "body" essentially the contents of the document

How to Add CSS to an HTML File

Adding CSS style to an HTML file can help make coding long HTML documents easier

One of the main purposes of using CSS is so that you can avoid repitition and lengthy code in HTML.

The reason avoiding using so much repitition is important is that in increases the chance of error. It's essentially using one standardized rule that can apply to multiple elements that only needs to be defined once.

If you had to write the same style conditions over and over, you increase the chance that you'll create errors along the way.

CSS also helps you avoid unnecessary duplication, it makes maintaining a website or page much easier, and you can make simple site-wide changes in one single place.

There are many, many different options for styling in CSS

Things that can be added with CSS style are color, font size, font family, etc

CSS can be added inline, in the header of the doc or in a separate .css file
